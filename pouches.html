What NOT to buy
  • Peak Design Small Tech Pouch — 16+ compartimentos accordion = DEEP cable management. Para tu vector (sparse o medium) es overengineering innecesario. Bellroy cubre lo mismo con menos saturación.
  • Tom Bihn Snake Charmer (Small o Large) — el costo real lo mata. Desglose Large: $38 producto + $14 DHL + €10 IVA España + €15 handling fee DHL en aduana = ~€73 puerta BCN. Único path <€55 = mule (amigo viajando US→ES) o eBay UK second-hand. Couriers tipo Stackry/MyUS NO sirven single-item.
  • Eagle Creek Pack-It Reveal Set — premium cubes innecesarios. Forclaz Lote 3 a €9.99 cubre todo a 5× menos costo.
  • Forclaz 2×15L / 2×7L — innecesarios. El Lote 3 ya incluye 1 cube grande de 12L + 2 chicos de 5.4L.
  • Compression cubes (Thule etc.) — community consensus: no comprimen lo suficiente para justificar el bulk extra.
  • Bagsmart / Amazon Basics no-name cubes — community warning: zippers fallan, sausage shape después de 6 meses.
  • Aer Slim Pouch / Wandrd Tech Bag — slim flat innecesario para setup tech-first en main del Recon.
  • Topo Designs Small / Micro Accessory Bag — €16-20 redundante teniendo los cubes chicos del Lote 3.
  • Bellroy Cinch Pouch / Standing Pouch — descartado / descontinuado.
  • Alpaka Tech Case Mini — €49 de Singapur, aduanas inciertas en checkout.
  • Vanguard Vesta Aspire / Solognac prismáticos — son cámara-dedicated, no tech pouch versátil.
  • Forclaz Ultralight / Trekking neceser — son neceseres de toiletry (ya tenés).
  • TRAVELHAWK organizador cables — €18.95 sin padding, mal ratio.
  • Evergoods CIVIC Access Pouch — €90+ puerta US, mismo problema courier que Tom Bihn.
  • Peak Design Wash Pouch — toiletry, ya tenés.
